# talktomycv

Mi CV como grafo de conocimiento interactivo — [en vivo](https://talktomycv.vercel.app).

- **Grafo**: canvas 2D con simulación de fuerzas propia (repulsión, muelles, gravedad, deriva ambiental), sin librerías.
- **Datos**: nodos tipados (roles, proyectos, skills, dominios) y relaciones con significado — los mismos principios que uso en knowledge bases reales.
- **Stack**: un único `index.html` autocontenido. Sin build, sin dependencias, sin cookies, sin analítica.
- Próximamente: mi versión digital — un agente sobre la API de Claude con coste por conversación a la vista.
